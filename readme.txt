fav-page - головний клас
при виборі складності на головний клас вішаю клас відповідний по назві
_easy - низька складність
_medium - середня складність
_hight - висока складність

в js є функція toggleBlocks, вона відповідає за всю логіку анімацій зміни сторінки коли юзер обрав складність
toggleBlocks(choseBlock, "choseHide", resultBlock, "resultShow", state) - приклад виклику, всі аргументи незмінні при зміні складності окрім state
туди текстом треба передавати один із трьох класів які відповідають за обрану складність, яку складність передамо така і відобразиться.

стилі блоку welcome залежать від fav-page, має чи ні він клас що відповіда за складність, і відображають відповідний контент.
